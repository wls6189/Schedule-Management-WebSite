  다른 사용자가 웹사이트에 접속하는 방법 (간단 요약)
═══════════════════════════════════════════════════════════════

⚠️ 중요: 다른 사용자는 터미널, Node.js, 코드 설치가 전혀 필요 없습니다!
         웹 브라우저만 있으면 됩니다!

───────────────────────────────────────────────────────────────
방법 1: Render 배포 주소 사용 (인터넷 어디서나 접속 가능)
───────────────────────────────────────────────────────────────

1. 웹 브라우저 열기 (Chrome, Edge, Firefox 등)

2. 주소창에 URL 입력
   예: https://schedule-management.onrender.com
   (서버 관리자에게 받은 주소)

3. Enter 키 누르기

4. 로딩 대기 (처음 접속 시 30초~1분 소요 가능)

5. 로그인하기
   - "사용자 이름 입력" 필드에 이름 입력
   - "로그인" 버튼 클릭
   - 완료!

───────────────────────────────────────────────────────────────
방법 2: 로컬 네트워크 주소 사용 (같은 Wi-Fi 필요)
───────────────────────────────────────────────────────────────

1. 서버 관리자와 같은 Wi-Fi에 연결

2. 웹 브라우저 열기

3. 주소창에 IP 주소 입력
   예: http://192.168.219.103:3000
   (서버 관리자에게 받은 주소)

4. Enter 키 누르기

5. 로그인하기
   - "사용자 이름 입력" 필드에 이름 입력
   - "로그인" 버튼 클릭
   - 완료!

───────────────────────────────────────────────────────────────
스마트폰에서 접속하기
───────────────────────────────────────────────────────────────

1. 브라우저 앱 열기 (Chrome, Safari 등)

2. 주소 입력 (Render 또는 로컬 주소)

3. 로그인 및 사용

───────────────────────────────────────────────────────────────
자주 묻는 질문
───────────────────────────────────────────────────────────────

Q: Node.js 설치가 필요한가요?
A: 아닙니다! 브라우저만 있으면 됩니다.

Q: 코드를 다운로드해야 하나요?
A: 아닙니다! 주소만 입력하면 됩니다.

Q: 비밀번호가 필요한가요?
A: 아닙니다! 이름만 입력하면 됩니다.

Q: 여러 명이 동시에 사용할 수 있나요?
A: 네! 제한 없이 동시 사용 가능합니다.
